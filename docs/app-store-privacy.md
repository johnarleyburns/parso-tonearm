# App Store Privacy Label

Tonearm has no account system, ads, analytics, telemetry, or tracking.

## Data Not Collected

- Contact info
- Health and fitness
- Financial info
- Location
- Contacts
- User content uploaded to Tonearm
- Browsing history
- Search history
- Identifiers for tracking
- Usage data
- Diagnostics
- Other data

## Data Used Only On Device Or User-Selected Services

- Music library metadata, playlists, favorites, play history, artwork, EQ settings, and sync identifiers stay on device unless the user enables iCloud sync. iCloud sync uses the user's Apple account and private CloudKit database.
- Remote library credentials and OAuth tokens are stored in Keychain and are sent only to the provider the user connected.
- Streaming cache audio is temporary local cache data and is never uploaded by Tonearm.

## Network Disclosure

Tonearm contacts only:

- archive.org sources the user adds.
- Apple's iTunes Search and mzstatic artwork CDN when missing-artwork lookup is enabled.
- User-connected remote libraries: Subsonic/Navidrome, Jellyfin, Plex, WebDAV, SMB, Dropbox, Google Drive, OneDrive, and pCloud.

No data is sold, shared for advertising, or used for cross-app tracking.
