PARSO · TONEARM · DESIGN MOCKUPS · V2

Tonearm — you bring the records

A local-first iOS player with archive.org sources added by URL. v2 removes downloads entirely: IA audio is stream-only, backed by a transparent LRU cache (default 500 MB, user-adjustable) that makes recently played music effectively offline without a download button ever existing. Local music arrives via Add Folder as Playlist or Add Audio Files. IA items, public user lists/playlists, favorites pages, and collections are all accepted as pasted URLs. No IA search, ever.

Changed from v1: Downloads tab → Playlists tab · download cloud icons → passive cache dots (empty ○ / filling ◔ / cached ●) · Add Source accepts list & favorites URLs · new Add menu, folder-import sheet, and cache settings screens · renamed Phono → Tonearm.
